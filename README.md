# Pentomino
伤脑筋十二块解法  
  
游戏规则参见：
https://baike.baidu.com/item/伤脑筋12块/10503954  

相传，伤脑筋12块的祖先是中国的骨牌，十二块不同形状的拼块可以拼出上千种形状，涉及数学中的几何学、拓扑学、运筹学、图论等多门学科，西方称为“中国的难题”或“潘多米诺骨牌”。“潘多米诺”是五子相连的意思，因为十二块拼板，虽说形状各异，但是每块所占的正方形之数相同，均为五个，因而得名。  
  
曾是漫画师丰子恺誉为“超平玩具之上，与象棋、国棋相颉”的“伤脑筋十二块”，是一种随心所欲、变幻无常的益智拼析游戏。 在二十世纪四十年代里“伤脑筋十二块”受到西方教学家们垂青，他们极力加以提倡，一时风靡全球。五十年代初，我国上海的一位中学教员方不圆教师，将流行的平面的“伤脑筋十二块”改造成立体的，使得“十二块”更伤脑筋，更富迷人的魅力。  
十二块不同形状的木块，把它从盒子中倒出来之后，很难装进去，真伤脑筋，不信你试试。
